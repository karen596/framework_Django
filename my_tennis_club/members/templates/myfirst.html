Que es Django?

Django es un framework de python que le permite crear de una manera más clara los sitios web utilizando python.

Funcion principal de Django

Desarrollo rapido: Django te permite crear sitios web de manera rápida y fácil.

Seguridad: Django te permite crear sitios web seguros.

Escalabilidad:Diseñado para manejar desde una aplicación a una base de datos de gran tamaño.

Estructura de Django

Django utiliza el patron de diseño "MVT (Modelo Vista Plantilla)"

El modelo: contiene los datos de una base de datos

La vista: controla las solicitudes que devuelve la plantilla y el contenido menos apropiado segun el modelo

La plantilla: es un archivo html que tiene como fin obtener y crear el diseño de la pagina web

URL: A parte Django proporciona una forma de navegar por las diferentes paginas de un sitio web

Ventajas de Django

Es un framework de código abierto

Es fácil de aprender

Es muy fácil de extender

Desventajas de Django

Overhead: Puede ser excesivo para aplicaciones muy simples

Puede limitar la flexibilidad de la aplicación

Puede usar mas recursos que frameworks mas ligeros
